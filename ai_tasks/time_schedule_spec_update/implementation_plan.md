# [仕様書更新] TimeSchedule 現行仕様の反映

TimeSchedule 機能における最近の大きな変更（データソースの `events` 配列カラムへの移行、オプション時間の導入、および詳細モーダルのレスポンシブ表示）を仕様書に反映します。

## ユーザーレビューが必要な事項

- **運用終了時刻の確認**: 
  - 現在の仕様書では 20:00 となっていますが、ソースコード (`timeScheduleService.js`) では 22:00 に延長されています。今回の更新ではコードの実装に合わせて 22:00 と記載しますが、運用上の意図と異なる場合はご指摘ください。

## 提案される変更

### ドキュメント

#### [MODIFY] [プロジェクト仕様書_TimeSchedule.md](file:///home/sho16/ikomasai-erp-2026/docs/プロジェクト仕様書_TimeSchedule.md)

- **データ構造の刷新**:
  - `event_schedule_slots` テーブルへの依存を廃止し、`events` テーブルの配列カラム（`schedule_dates`, `schedule_start_times` 等）を参照する仕組みに変更されたことを明記します。
- **オプション時間（準備・片付け）の追加**:
  - 企画ごとに「準備時間」「片付け時間」が設定可能になり、詳細モーダルで表示される仕様を追加します。
- **レスポンシブ表示と時刻整列**:
  - 詳細モーダルにおけるオプション時間のスマホ・PC別レイアウト、および `tabular-nums` を用いた数字の垂直整列について追記します。
- **運用時間の更新**:
  - 運用終了時刻を実装に合わせて 22:00 へ更新します。
