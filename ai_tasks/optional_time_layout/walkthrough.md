# 開催日程のオプション時間表示レイアウト修正：完了報告

企画詳細モーダルにおける開催日程の表示レイアウトを、スマホ・PC両対応のレスポンシブな形式に修正し、視認性を大幅に向上させました。

## 変更内容

### [DetailModal.jsx](file:///home/sho16/ikomasai-erp-2026/src/features/01_Events&Stalls_list/components/DetailModal.jsx)

- **レスポンシブ・レイアウト**:
  - **PC（600px以上）**: 準備時間と片付け時間を横並び（セパレーター「／」付き）で表示。
  - **スマホ（600px未満）**: 準備時間と片付け時間を縦にスタック（2行）表示し、重なりを解消。
- **時刻表示の精緻化 (`FormattedTime`)**:
  - `fontVariant: ['tabular-nums']` を導入。数字が等幅になり、異なる時刻でも開始/終了位置がピタリと揃います。
  - コロン（:）の垂直位置を微調整。数字の中央高さに合わせることで、洗練された印象を与えます。
  - `includeFontPadding: false` により、OSやデバイスによるフォント由来の不要な余白を排除しました。

## 検証結果

- **レスポンシブ動作**: ブラウザの幅を変更し、レイアウトが動的に（PC: 横、スマホ: 縦）切り替わることを確認。
- **視認性**: `tabular-nums` により数字の列が綺麗に揃い、コロンの位置調整によってデジタル時計のようなプロフェッショナルな外観になっていることを確認。
- **配置**: 開催時間の「数字の頭」と、オプション時間の開始位置が垂直に揃っていることを確認。
