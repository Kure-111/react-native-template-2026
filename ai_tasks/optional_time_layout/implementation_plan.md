# [企画詳細モーダル] 準備・片付け時間のレスポンシブレイアウト修正

企画詳細モーダルにおいて、準備時間と片付け時間がスマホで重なって表示される問題を修正し、PCでは横並び、スマホでは縦並び（スタック）になるレスポンシブレイアウトを導入します。また、等幅数字（tabular-nums）の採用とコロンの垂直位置微調整により、視認性とプロフェッショナルな外観を追求します。

## ユーザーレビューが必要な事項

> [!IMPORTANT]
> - 画面幅 600px を閾値として、スマホ表示（縦並び）とPC表示（横並び）を切り替えます。
> - 数字の「ズレ」を解消するため、等幅フォント設定（`tabular-nums`）を適用しました。

## 提案される変更

### [ikomasai-erp-2026](file:///home/sho16/ikomasai-erp-2026)

#### [MODIFY] [DetailModal.jsx](file:///home/sho16/ikomasai-erp-2026/src/features/01_Events&Stalls_list/components/DetailModal.jsx)

- **レスポンシブ対応**: `useWindowDimensions` を使用して画面幅を取得。
- **条件付きレンダリング**:
    - PC（>= 600px）: 準備時間と片付け時間を `flexDirection: 'row'` で並べ、間に「／」セパレーターを表示。
    - スマホ（< 600px）: 準備時間と片付け時間を `flexDirection: 'column'` で積み上げ、セパレーターを非表示に。
- **時刻表示の最適化 (`FormattedTime` コンポーネント)**:
    - `fontVariant: ['tabular-nums']` を指定し、数字の幅を固定。
    - 時刻とコロン（:）を個別の `Text` 要素に分解。
    - コロンの `paddingBottom` を調整し、数字の中央高さに合わせる。
    - `includeFontPadding: false` で不要な余白を排除。

## 検証計画

### 自動テスト
- なし（UIレイアウトの微調整のため）

### 手動確認
- **PC表示**: ブラウザの幅を広げ、準備・片付け時間が横並びでセパレーターが表示されることを確認。
- **スマホ表示**: ブラウザを狭めるかデバイスモードを有効にし、準備・片付け時間が縦に並び、セパレーターが消えることを確認。
- **文字の揃い**: 時刻の数字がガタつかず、垂直に揃っていることを確認。
